# AI-CHATBOT-WITH-NLP

This project presents the development of an intelligent AI chatbot built using Python, integrating Natural Language Processing (NLP) techniques to simulate human-like conversations. The chatbot was designed and implemented as part of an internship task, with the primary objective of understanding how machines can process, interpret, and respond to human language effectively. The development process began with setting up the required environment, where essential libraries such as NLTK and spaCy were installed and configured. NLTK was used for fundamental text preprocessing tasks including tokenization, stopword removal, and normalization, while spaCy was utilized to compute semantic similarity between user queries and predefined responses, enabling the chatbot to go beyond simple keyword matching.

Initially, a rule-based chatbot was implemented to establish the foundation of the system. This involved creating a structured knowledge base consisting of multiple user queries and corresponding responses. The chatbot was then enhanced with an interactive loop, allowing continuous user interaction until an exit command was provided. In the next stage, NLP techniques were incorporated to improve the chatbot’s understanding of user input. The preprocessing pipeline ensured that unnecessary words, punctuation, and noise were removed, thereby improving the accuracy of intent recognition. Using spaCy’s similarity function, the chatbot compares the processed user input with stored queries and selects the most relevant response based on a defined similarity threshold.

To further enhance the chatbot’s performance and user experience, several advanced features were added. Greeting detection was implemented to provide instant and natural responses to common salutations such as “hello” or “hi.” Multiple responses were introduced for each query, allowing the chatbot to respond dynamically and avoid repetitive answers, making the interaction feel more human-like. Basic context handling was achieved using session-based memory, enabling the chatbot to maintain and display conversation history within the interface. Additionally, a simple spell correction mechanism was integrated using Python’s difflib library, allowing the chatbot to handle minor user input errors and improve robustness.

A major upgrade in this project was the development of a graphical user interface using Streamlit. This transformed the chatbot from a command-line application into a fully interactive web-based application. The Streamlit interface provides a clean and intuitive design, where users can input their queries and view responses in a conversational format. The chat history feature enhances usability by displaying previous interactions, thereby improving engagement and continuity. This UI integration significantly elevates the project’s presentation, making it suitable for real-world applications and portfolio demonstration.

For project organization and reproducibility, a requirements.txt file was created to list all necessary dependencies, ensuring that the project can be easily set up and executed in different environments. Overall, this project demonstrates a comprehensive understanding of chatbot development, NLP fundamentals, and user interface design. It showcases the ability to combine multiple technologies to build a functional and intelligent system, making it a valuable addition to any software development or data science portfolio.
